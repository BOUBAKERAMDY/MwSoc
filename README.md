Let'sGoBiking -- Launch Instructions


## ✅ 1. Launch All Components

Once ActiveMQ is running, go to the root folder and **run the launcher
script**:

    RUN-ALL.bat


Right‑click the file and select:

**➡️ "Run as administrator"**

This script will automatically start: - the **Proxy Cache Server** - the
**Routing Server** - the **Heavy Client** and also the **front**
